## ðŸŽ¨ How to Use

### **For Selected Notes:**
1. Enable **Movable** mode (`Edit > Movable` or `Ctrl+D`)
2. Select one or more buttons:
   - Drag to create selection rectangle
   - `Ctrl+A` to select all
   - Click individual buttons while holding `Ctrl`
3. Go to `View > Styles` and choose any style
4. **Selected buttons transform** with the new style!

### **For All Notes:**
1. Simply go to `View > Styles` without selecting anything
2. Choose a style
3. **All notes** will be styled uniformly
